# Sobre o projeto

Chat simples criado para testar e desenvolver habilidades com React js. 

# Tecnologias utilizadas

- React.js
- Firebase

# Como executar o projeto

```bash
# Clonar repositório
  git clone https://github.com/jaovsb/chat.git

# Instalar dependências
  yarn install
  
# Iniciar projeto
  yarn start
```
